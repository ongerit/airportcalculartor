# airport calculator
Prototype for Calculating distance between airports
## Requirements:

node >= 4.0.0
gulp >= 3.9.0 w/ external module babel-core/register



## Installation

```bash
$ cd airportcalculator
$ npm install
$ bower install
```

## Configuration

All build config is stored in gulpfile.babel.js


## Develop
```bash

$ gulp serve
```

## Build
```bash

$ gulp build
```
